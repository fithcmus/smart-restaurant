# MVP Scope

Version: 0.1
Date: 2025-10-31
Author:

## In-scope (MVP)
- Tenant signup & onboarding (basic)
- QR per table -> customer ordering (view menu, add to cart, checkout)
- Menu management (create items, categories, prices)
- Order processing for staff (KDS or simple order list)
- Basic payments (card or external POS integration stub)

## Out-of-scope (not in MVP)
- Advanced promotions/loyalty
- Multi-location analytics
- Offline mode for kiosks

## Acceptance boundary
- Mobile-first ordering experience must work on modern browsers (Chrome, Safari)
- Admin features accessible via desktop web
