# ADR Index

This folder contains Architecture Decision Records (ADRs). Each ADR documents a significant architectural or strategic decision:
- the problem or context
- considered options
- the chosen option and rationale
- consequences and migration notes

ADRs are numbered using the pattern `0001-<short-title>.md`.
